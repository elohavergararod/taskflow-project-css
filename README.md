# TaskFlow

## 📌 Description

TaskFlow is a simple and intuitive task management web application that allows users to create, filter, and manage tasks efficiently.

## 🚀 Features

* Add new tasks with priority and category
* Filter tasks by status (All, Pending, Completed)
* Filter by priority (High, Medium, Low)
* Filter by category (Work, Studies, Personal)
* Search tasks in real time
* Task counter display

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (Vanilla)

## ⚙️ Installation

Clone the repository and open the project:

```bash
git clone https://github.com/USERNAME/taskflow-project-css.git
cd taskflow-project-css
```

Then open `index.html` in your browser.

## ▶️ Usage

1. Type a task in the input field
2. Select priority and category
3. Click **+ Add**
4. Use filters to organize your tasks
5. Search tasks using the search bar
